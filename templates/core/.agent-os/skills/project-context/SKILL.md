# Project Context

Use this skill when a task requires understanding the existing project before editing.

## Workflow

1. Inspect top-level files, package metadata, and relevant docs.
2. Locate the files directly related to the task.
3. Identify existing patterns before proposing new structure.
4. Keep the gathered context focused on the requested work.
