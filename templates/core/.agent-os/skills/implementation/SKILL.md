# Implementation

Use this skill when editing project files.

## Workflow

1. Check the worktree before editing.
2. Read target files and nearby patterns.
3. Make the smallest change that satisfies the request.
4. Do not revert unrelated user changes.
5. Keep new abstractions justified by real complexity or reuse.
