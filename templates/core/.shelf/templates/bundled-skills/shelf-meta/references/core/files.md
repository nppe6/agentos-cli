# AgentOS Shelf File Reference

This page describes the current default files generated by `agentos-cli shelf init`.

## `.shelf/`

```text
.shelf/
|- .developer
|- .runtime/
|- .gitignore
|- config.yaml
|- manifest.json
|- template-hashes.json
|- update-manifest.json
|- workflow.md
|- scripts/
|- skills/            # Optional project-local custom skills
|- agents/            # Optional project-local agent overrides
|- rules/             # Optional local rule override sources
|- spec/
|- tasks/
\- workspace/
```

| Path | Purpose |
| --- | --- |
| `.shelf/workflow.md` | Shared workflow source of truth. |
| `.shelf/config.yaml` | Project configuration and task lifecycle hook configuration. |
| `.shelf/manifest.json` | Records selected tools, generated files, and CLI version. |
| `.shelf/template-hashes.json` | Lets update/sync detect user-modified generated files. |
| `.shelf/.developer` | Local developer identity. |
| `.shelf/.runtime/` | Session-scoped runtime state, including active task pointers. |
| `.shelf/spec/` | Project-specific engineering specs. |
| `.shelf/tasks/` | Task PRDs, JSONL context, research, and archives. |
| `.shelf/workspace/` | Developer journals and cross-session memory. |
| `.shelf/scripts/` | Python runtime scripts used by CLI wrappers and workflow commands. |
| `.shelf/skills/` | Optional project-local custom skills that can be projected to supported tools. |
| `.shelf/agents/` | Optional project-local agent overrides projected to supported tools. |
| `.shelf/rules/` | Optional local managed-entry override sources such as `AGENTS.shared.md`. |

## Platform Files

| Path | Purpose |
| --- | --- |
| `AGENTS.md` | Root shared rules entry for the current generated platforms. |
| `.agents/skills/shelf-*` | Shared skills for Codex/Open Agent Skills. |
| `.codex/skills/*` | Codex-specific local skills. The current default projection creates the directory but does not duplicate shared `shelf-*` skills into it. |
| `.codex/agents/shelf-*.toml` | Codex agent definitions. |
| `.codex/config.toml` | Project-scoped Codex defaults. |
| `.codex/hooks.json` and `.codex/hooks/shelf-*.py` | Codex hook wiring and Shelf context breadcrumbs. |
| `.agents/skills/shelf-continue/SKILL.md` | Codex-readable shared skill projected from the common `continue` command source. |
| `.agents/skills/shelf-finish-work/SKILL.md` | Codex-readable shared skill projected from the common `finish-work` command source. |
| `.claude/skills/shelf-*` | Claude Code skill copies. |
| `.claude/agents/shelf-*.md` | Claude Code agent definitions. |
| `.claude/commands/shelf/*.md` | Claude Code slash command prompts. |
| `.claude/settings.json` | Registers the Claude Code session-start hook. |
| `.claude/hooks/shelf-session-start.py` | Lightweight startup reminder. |
| `.claude/hooks/shelf-inject-workflow-state.py` | Per-turn workflow-state breadcrumb hook. |

## Runtime State

Active task state lives in `.shelf/.runtime/sessions/<session-key>.json`.

`.shelf/.current-task`, `.shelf/.ralph-state.json`, `.shelf/worktree.yaml`, worktree registries, and quality-loop state files are not part of the current default install. Treat them as custom local files if they exist.

## Update Behavior

`agentos-cli shelf update` uses `manifest.json`, `template-hashes.json`, backups, and `.shelf/update.skip` to update generated files without overwriting user-modified content by default.
