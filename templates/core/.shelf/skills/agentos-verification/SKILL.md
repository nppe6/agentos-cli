# AgentOS Verification

Use this skill before reporting completion.

## Workflow

1. Prefer targeted tests, type checks, lint checks, or manual inspection tied to the changed files.
2. Run broader checks when shared contracts, build config, routing, state, or public APIs are affected.
3. Report commands run, failures, blockers, and residual risk.
