# AgentOS Documentation

Use this skill when a finding should guide future agents or teammates.

## Workflow

1. Record durable context, not transient chat history.
2. Include when the guidance applies, what to do, and how to verify it.
3. Avoid sensitive data, credentials, or one-off implementation noise.
