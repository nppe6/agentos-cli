# AgentOS Debugging

Use this skill for errors, regressions, failing tests, or unclear broken behavior.

## Workflow

1. Reproduce or localize the symptom when possible.
2. Trace from observable failure to root cause.
3. Prefer a focused fix over broad rewrites.
4. Add or update verification when the fix changes behavior.
