# AgentOS Planning

Use this skill for non-trivial changes, ambiguous requests, or work that affects multiple files.

## Workflow

1. Clarify the goal, non-goals, affected surface, and acceptance checks.
2. Split the work into small verifiable units.
3. Note risks, dependencies, and rollback concerns.
4. Avoid implementation until the scope is clear enough to execute safely.
